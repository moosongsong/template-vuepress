# template-vuepress

손쉽게 github pages로 배포하는 vuepress 템플릿입니다!

배포를 하기 위해서는 secrets 에 ACCESS_TOKEN이 반드시 등록되어 있어야 합니다.

root / 위치의 Readme.md는 배포시 첫 화면입니다.

# local에서 테스트하기

yarn dev

vuepress는 yarn을 권장하고 있습니다.

# 빌드하기

yarn build
